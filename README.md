# 🚚 SQL Window Functions Practice Project: Logistics & Supply Chain Analytics

## 📌 Overview

The project demonstrates how SQL Window Functions can solve complex business problems involving shipment analysis, warehouse operations, customer segmentation, carrier performance, and logistics reporting.

---

# 🎯 Project Objectives

This project aims to:

* Practice advanced SQL Window Functions
* Solve real-world logistics and supply chain business problems
* Improve analytical SQL skills
* Strengthen understanding of CTEs and Window Functions
* Build a portfolio-ready SQL project for GitHub

# 📈 Learning Outcomes

After completing this project, you will be able to:

* Use SQL Window Functions confidently
* Build complex analytical queries
* Perform cumulative calculations
* Create ranking reports
* Segment customers using `NTILE()`
* Solve real-world logistics analytics problems
* Apply SQL to supply chain business scenarios
